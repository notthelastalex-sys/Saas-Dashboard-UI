# Modern SaaS Dashboard UI

A modern responsive SaaS dashboard interface built using HTML and CSS.
This project showcases a professional admin dashboard layout with analytics cards, project management sections, responsive tables, and a modern dark UI.

---

## Features

- Responsive dashboard layout
- Sidebar navigation menu
- Analytics/stat cards
- Project overview cards
- Responsive data table
- Modern dark UI design
- Mobile responsive layout
- Clean card-based interface

---

## Technologies Used

- HTML5
- CSS3
